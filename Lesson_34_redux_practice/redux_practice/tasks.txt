1. Создать три страницы: MainPage, UsersPage, PostsPage. На главной странице создать две кнопки, по клику на которые нас перебрасывает на страницу с юзерами или постами

2. Установить redux
npm i redux react-redux

3. Создать хранилище, корневой reducer, обернуть приложение в Provider

4. Внутри UsersPage создать форму с тремя инпутами - Name, Age, Country. При отправке формы собирать данные и выводить в консоль.

5. Создать userReducer, action addUser. В userReducer прописать функцию добавления юзера. Добавить userReducer в корневой reducer

6. Достать state и создать из него карточки User
