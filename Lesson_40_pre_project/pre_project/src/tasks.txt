1. Создать categoriesReducer
2. Создать action loadCategories, прописать для него логику в редьюсере
3. Написать fetch get запрос для получения списка всех категорий
4. Записать полученный ответ с категориями в состояние categories

5. Написать функцию, которая отправляет fetch-запрос на продукт. Вывести информацию о продукте в консоль

6. Создать productReducer
7. Записать в состояние то, что придет по запросу. Проверить через консоль записалось ли что-то в состояние
8. Вывести на экран title, image, description, price, кнопка
9. При клике на карточку товара перебрасывать на страницу с описанием этого товара
