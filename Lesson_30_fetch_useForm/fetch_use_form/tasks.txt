1. Создать компонент NavMenu с двумя ссылками - на страницу с пользователями и на страницу с постами
2. Настроить переключение между страницами по нажатию на ссылки в меню
3. В отдельном файле прописать функцию getUsers, отправляющую fetch-запрос (get) по адресу https://dummyjson.com/users и выводящую в консоль массив users
4. Импортировать функцию getUsers в App, вызвать ее в App. Проверить выводится ли в консоль информацию о пользователях
5. Создать переменную состояния users и функцию setusers, отвечающую за отслеживание изменения состояния. Поместить данные, полученные из запроса, в состояние users.
6. Настроить отправку fetch-запроса при вызове функции getUsers() один раз при обновлении компонента.
7. Создать компоненты User и UsersContainer
8. Импортировать UsersContainer в UsersPage
9. Из массива с данными о пользователях для каждого пользователя создать компонент User в UsersContainer. Карточка пользователя должна содержать параграфы Firstname, Lastname, Age. Используйте Context
10. Создать компонент AddUserForm с тремя инпутами (Firstname, Lastname, Age), импортировать его в UsersPage.
11. Научить форму собирать данные и выводить их в консоль при отправке формы.
